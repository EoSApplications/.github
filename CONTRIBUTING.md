# Contributing

The public repositories under the EoSApplications organization do not accept external code contributions. Every change is reviewed and merged by the organization owners.

If you've found a bug, have a question, want to suggest a calibration addition, or want to report a calibration error please email:

- Allison Pease: apease13@asu.edu
- Heidi N. Krauss: Heidi.N.Krauss@gmail.com

For security vulnerabilities specifically, see `SECURITY.md` instead of emailing the addresses above.

This file is the organization-wide default, used by any EoSApplications repository that doesn't have its own `CONTRIBUTING.md`.
